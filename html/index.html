Browser image framing tool

画像を切り抜かず、余白で比率を整える

FrameMaker は、画像の外側に余白フレームを追加して、SNS投稿、ブログ用画像、YouTubeサムネイル、 ポートフォリオ画像、スクリーンショット整理に使いやすい形へ整えるブラウザツールです。 画像処理はブラウザ上で行い、通常の利用では加工対象の画像をサーバーへ送信しません。

使いどころ

SNS投稿画像、ブログのアイキャッチ、YouTubeサムネイル、作品一覧、アプリ画面のスクリーンショット整理に使えます。

比率の選び方

正方形はSNS向け、4:3は説明画像向け、黄金比や平方根3は横長の見せ方に向いています。

画像の扱い

選択した画像はブラウザ内でプレビューと保存を行います。Poppoworks側で画像を保存する設計ではありません。

FrameMakerは無料で使えますか？

無料で利用できます。作成した画像は、元画像の権利や利用条件を確認したうえで使用してください。

広告はどの画面に表示されますか？

AdSenseの自動広告設定により、ページ本文や説明コンテンツの周辺に広告が表示される場合があります。画像の保存操作を妨げない配置になるよう、ページ構成を調整しています。

詳しい説明はどこで読めますか？

PoppoworksのFrameMaker紹介ページで、用途、比率の使い分け、更新履歴を掲載しています。
